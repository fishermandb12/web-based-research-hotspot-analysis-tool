# 1区期刊筛选功能说明

## 🎯 功能说明

**"识别1区期刊"功能的作用：**
- ✅ 使用 LLM 识别领域内的顶级期刊（中科院1区/SCI Q1）
- ✅ **只保留**来自这些期刊的论文
- ✅ **过滤掉**其他期刊的论文

## 📊 工作流程

### 完整流程示例

```
1. 输入关键词：经济学
   ↓
2. 勾选"识别1区期刊"
   ↓
3. LLM 识别1区期刊
   → 识别到：Journal of Political Economy, 
              American Economic Review, 
              Quarterly Journal of Economics 等
   ↓
4. 从 OpenAlex 获取 100 篇论文
   ↓
5. 筛选期刊
   → 保留：20篇来自1区期刊的论文
   → 过滤：80篇来自其他期刊的论文
   ↓
6. 提取关键词（只从20篇1区论文中提取）
   ↓
7. 生成热力图
```

## 💡 为什么要过滤？

### 优势

**1. 确保论文质量**
- ✅ 1区期刊是领域内最顶级的期刊
- ✅ 论文质量高，审稿严格
- ✅ 代表领域最前沿的研究

**2. 提高关键词准确度**
- ✅ 顶级期刊的关键词更规范
- ✅ 研究方向更明确
- ✅ 热点识别更准确

**3. 聚焦核心研究**
- ✅ 避免低质量论文干扰
- ✅ 关注真正的研究热点
- ✅ 反映领域主流方向

### 劣势

**1. 论文数量减少**
- ⚠️ 从100篇减少到20篇
- ⚠️ 可能遗漏一些新兴方向
- ⚠️ 数据量较小

**2. 可能过于保守**
- ⚠️ 新兴期刊被排除
- ⚠️ 跨学科研究可能被忽略
- ⚠️ 区域性研究可能缺失

## 🔧 调整建议

### 如果过滤后论文太少

**问题：**
```
从 100 篇论文中筛选出 5 篇来自1区期刊的论文（过滤了 95 篇）
```

**原因：**
1. 该领域1区期刊较少
2. 时间范围太短
3. 关键词太具体

**解决方案：**

**方案 1：扩大时间范围**
```
从：2023-2024（2年）
改为：2020-2024（5年）
```

**方案 2：使用更通用的关键词**
```
从：量子纠错码
改为：量子计算
```

**方案 3：不勾选期刊筛选**
```
☐ 识别1区期刊
→ 分析所有100篇论文
```

### 如果想要更多1区论文

**当前限制：**
- OpenAlex 每次最多返回 100 篇论文

**未来改进：**
- 可以增加到 200-500 篇
- 分批获取论文
- 合并结果

## ❓ 常见问题

### Q1: 为什么过滤了这么多论文？

**A:** 这是正常的。1区期刊是顶级期刊，数量本来就少。在100篇论文中，可能只有10-30篇来自1区期刊。

### Q2: 过滤掉的论文是不是浪费了？

**A:** 不是浪费，而是**精准筛选**。目的是只分析最高质量的论文，确保关键词和热点的准确性。

### Q3: 可以看到被过滤的论文吗？

**A:** 当前版本不显示被过滤的论文。如果需要，可以：
1. 不勾选"识别1区期刊"
2. 查看所有100篇论文的分析结果

### Q4: 如何知道哪些期刊是1区？

**A:** 点击"📋 查看期刊列表"可以看到 LLM 识别的1区期刊名单。

### Q5: 可以自定义期刊列表吗？

**A:** 当前版本不支持。未来版本可能会添加：
- 手动输入期刊列表
- 编辑 LLM 识别的期刊
- 保存常用期刊列表
