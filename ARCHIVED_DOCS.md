## 主要变化对比

### v2.x → v3.0 的重大变化

| 特性 | v2.x | v3.0 |
|------|------|------|
| **API Key** | 可选 | **必填** |
| **关键词提取** | 规则提取 + 可选LLM | **仅LLM** |
| **Q1 期刊筛选** | 可选 | **默认启用** |
| **界面复杂度** | 多个选项 | **简化** |
| **结果质量** | 不一致 | **一致高质量** |
| **处理方式** | 批量或规则 | **逐篇LLM** |

### 为什么升级到 v3.0？

1. **一致的高质量**：所有用户都获得相同的优秀体验
2. **简化的流程**：更少的选择，更清晰的使用方式
3. **可靠的处理**：逐篇处理，单篇失败不影响整体
4. **增强的指导**：详细的错误信息和故障排查

## 迁移指南

如果您正在使用 v2.x 版本，请参考以下步骤升级到 v3.0：

### 步骤 1：获取 API Key
1. 访问 [阿里云 DashScope](https://dashscope.console.aliyun.com/)
2. 注册/登录账号
3. 创建 API Key
4. 复制 API Key

### 步骤 2：清理旧配置
```bash
# 删除 .env 文件（如果存在）
rm .env

# 清除缓存（在应用中点击"清除缓存"按钮）
```

### 步骤 3：启动并配置
```bash
streamlit run app.py
```

在侧边栏输入 API Key，开始使用 v3.0。
